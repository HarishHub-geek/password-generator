## âœ¨ Features

* **Generate Strong Passwords:** Creates highly secure passwords using a combination of uppercase letters, lowercase letters, numbers, and special characters.
* **Customizable Length:** Easily adjust the password length up to **18 characters** (or the maximum supported by the slider).
* **Inclusion Options:** Toggle the inclusion of **Numbers** and **Special Characters** to meet specific password requirements.
* **One-Click Copy:** Quickly copy the generated password to your clipboard with the **Copy** button.
